# VERITY-data

VERITY 의 frontend serving data snapshots. Auto-published from
[gywns0126/VERITY](https://github.com/gywns0126/VERITY) via publish-data action.

- **Read-only consumer** — direct push 금지
- **force_orphan main branch** — history 1개
- **Raw URL pattern**: `https://raw.githubusercontent.com/gywns0126/VERITY-data/main/<file>`
- **Frontend (Framer)**: portfolio.json / recommendations.json / macro_snapshot / price_pulse / 등 fetch (corp 자산주는 live API endpoint)
